# アプリコンテストまとめ

https://contest.pronama.jp/ のソースコードです。

## 注意

* ユーザー登録機能等は無効化しています。ユーザーや Role を作成しないと動作しません。
* <head> 内の Google 広告、Google アナリティクス、Font Awesome、Adobe Font および フッターの 忍者ツールズ のコードをコピーして使わないでください。
* Web ページのサムネイル作成機能はこのプロジェクトには含まれていません。動作もしません。サムネイル取得先の Azure Storage にアクセスしても新規に取得できません。
* メールの送信は SendGrid を使っています。API key が無ければ動作しません。
